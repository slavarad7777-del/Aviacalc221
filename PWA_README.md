# AvisCalc PWA

Файлы для GitHub Pages:

```text
index.html
app.css
app.js
manifest.webmanifest
service-worker.js
icons/
```

## Как обновить репозиторий

Загрузи в корень репозитория все файлы из этого архива и сделай commit.

## iPhone

1. Открой сайт в Safari.
2. Нажми кнопку «Поделиться».
3. Выбери «На экран Домой».
4. Запускай AvisCalc как приложение.

## Android

Открой сайт в Chrome и нажми «Установить приложение» или меню → «Добавить на главный экран».

## Оффлайн

После первого открытия service worker кэширует основные файлы:

- index.html
- app.css
- app.js
- manifest
- icons

Данные приложения продолжают храниться локально в браузере.
